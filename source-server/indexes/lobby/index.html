Brick Game

Each player should open the Gameboard in one browser window.

In a separate browser window each player should open the Player Controller (id provided by the admin) and all other players should grab a Brick Controller.

The object of the game is for the players with the Brick Controllers to send bricks down their assigned row to push the player off of the gameboard. The player with the Player Controller should, you know, not get pushed off the gameboard.
